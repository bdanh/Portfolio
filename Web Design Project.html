Overview

The Althist or Alternate History was the first page to be created and pretty much directed how the assignment was designed. The page was initally just an empty page with the could now be the "article" on it. As further assignements were given such as adding CSS and forms, the direction that the site when was one of a blog. Each element added with that in mind.

  1. The general layout of CSS was modeled on how blogs were usually designed with a long header containing a logo, space for the article and some form navigation.
  2. Eventually we were required to add forms to out sites for an assignment, one standard and another on existing page. For mine came in from of a "comments section" to the althist page and creation of the Suggestions page.
  3. The final addition was the index, here named the archive, as we required to host our sites on Github which requires an index to serve as the site's homepage. The archive came as a listing of an archive all existing articles.
